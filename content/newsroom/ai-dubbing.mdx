Machine dubbing is fast, cheap, and close enough for a lot of content. It is not close enough for live commentary, where a mistranslated offside call or a garbled player name breaks trust with an audience in real time.

We tested current dubbing tools against our own commentary output across several matches. The tools kept pace. They got tactical terminology, club nicknames and regional football slang wrong often enough that a native commentator was still the only safe option for anything live.

### Where AI dubbing does work

Post-match highlight packages, magazine content, anything with a review pass before publish. The technology is good enough there today. Live is a different job.
